# Mobile Gaming Optimizer

Mobile Gaming Optimizer is an open-source project that helps Android users improve gaming performance.

## Features
- FPS optimization tips
- Android developer settings for gaming
- Lag reduction methods
- Sensitivity setup guides

## Supported Games
- PUBG Mobile
- Free Fire
- Call of Duty Mobile

## Goal
Help mobile gamers optimize their devices for smoother gameplay and better FPS.

## Contribution
Anyone can contribute to improve gaming optimization guides.
